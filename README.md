# 🍔 FastBite

FastBite is a modern and responsive food ordering website developed using HTML, CSS, and JavaScript. The website allows users to browse restaurants, view menus, explore deals, and manage their food cart through an attractive user interface.

## 🚀 Features

* Responsive Navigation Bar
* Hero Section
* Restaurant Cards
* View Menu Popup
* Shopping Cart
* Quantity Buttons
* Remove Items
* Top Deals Section
* Mobile Responsive Design
* Smooth Scrolling

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome
